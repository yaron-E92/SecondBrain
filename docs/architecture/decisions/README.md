# Architecture Decision Records

Architecture Decision Records (ADRs) capture significant technical decisions for
SecondBrain. ADRs are numbered sequentially and are not renumbered after they are
accepted or superseded.

Each ADR uses the following sections:

- Status
- Context
- Decision
- Consequences
- Alternatives considered
- Follow-up decisions
